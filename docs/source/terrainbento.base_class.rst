Model Base Classes
==================

Presently terrainbento has three built in base classes. The **ErosionModel**
provides functionality used by all models while the **StochasticErosionModel**
provides functionality used by models that explicitly treat precipitation. The
**TwoLithologyErosionModel** reads in the lithology contact information required
for models with two lithologies.
